Política de Privacidade — CuteStickers Português (pt-BR) O CuteStickers respeita sua privacidade. Este aplicativo permite que usuários descubram imagens de cães e gatos, salvem imagens localmente em seus dispositivos e exportem figurinhas para aplicativos compatíveis. Dados coletados O CuteStickers não coleta dados pessoais identificáveis, como nome, e-mail, telefone ou localização. Dados de uso O aplicativo pode acessar: Conexão com a internet para carregar imagens de APIs públicas. Armazenamento local do dispositivo para salvar imagens e figurinhas geradas. Esses dados permanecem no dispositivo do usuário e não são enviados para nossos servidores. Compartilhamento com terceiros O aplicativo pode interagir com aplicativos de terceiros, como o WhatsApp, apenas quando o usuário solicitar exportar uma figurinha. Segurança Não armazenamos informações pessoais em servidores externos.
